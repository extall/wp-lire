=== Lire - LaTeX inspired Reference Extensions ===
Contributors: Alexei Teplyakov
Tags: tex, latex, reference, link, cross-reference, math, parser
Requires at least: 2.5
Tested up to: 3.3
Stable tag: 0.1

This plugin uses the LiRE library to create nice cross-referenced posts in your WordPress blog. Math support can be added using a plugin of preference, e.g. MathJAX.

== Description ==

Latex compatible syntax. See LiRE library description for reference.

== Installation ==

# Upload the `wp-lire` folder with all files to the `/wp-content/plugins` directory;
# Activate the plugin through the 'Plugins' menu in WordPress.
